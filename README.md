# sentiment-analysis--OIBSIP

This project focuses on analyzing customer reviews and classifying them into Positive, Negative, and Neutral sentiments using Microsoft Excel.

## 📌 Project Overview
This project focuses on analyzing customer reviews and classifying them into Positive, Negative, and Neutral sentiments using Microsoft Excel.

## 🎯 Objectives

- Understand customer feedback
  
- Identify sentiment trends
  
- Build an interactive Excel dashboard


## 🛠 Tools Used

- Microsoft Excel
  
- Pivot Tables
  
- Charts (Bar, Pie)
  
- Slicers

## 📊 Dashboard Features

- Sentiment Distribution (Pie Chart)
  
- Review Count by Category (Bar Chart)
  
- Filters using Slicers
  
- KPI Cards (Total Reviews, Positive %, Negative %)

## 🔄 Process

1. Data Cleaning
   
2. Data Preparation
   
3. Sentiment Categorization
   
4. Pivot Table Creation
   
5. Dashboard Design


## ✅ Conclusion

Excel can effectively be used for basic sentiment analysis and visualization without coding.
